# Демо-видео

Публичный репозиторий не содержит текста озвучки: голос записывается автором и является частью
готового ролика. Этот файл фиксирует только проверяемые технические границы записи.

Target: 168 seconds. Жёсткое требование конкурса: `<180` секунд, голосовая озвучка и полный
сквозной путь.

## Что должно быть видно

1. Создание кампании из редактируемого брифа и факт-карты.
2. Живой запуск Ouroboros и два MCP-инструмента: получение контекста и сохранение черновика.
3. СМС, письмо, доказательства фактов и автоматическая проверка качества.
4. Замечание редактора, новая версия и точный diff.
5. Предложение и человеческое утверждение ограниченного правила.
6. Применение правила к новой подходящей кампании.
7. Десять подтверждённых живых результатов, синтетические данные и отсутствие отправки.

Долгое ожидание ответа провайдера допускается сократить монтажом. На экране сохраняются один URL,
идентификатор запуска, режим `live_ouroboros` и подпись о сокращённом ожидании; сохранённый результат
нельзя выдавать за новый запуск.
